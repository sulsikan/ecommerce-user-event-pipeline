# Dataset

The raw ecommerce behavior CSV is not committed to this repository because it is
too large for normal Git hosting.

Download `2019-Oct.csv` from Kaggle and place it in this directory:

https://www.kaggle.com/datasets/mkechinov/ecommerce-behavior-data-from-multi-category-store?select=2019-Oct.csv

Expected local path:

```text
data/2019-Oct.csv
```

The architecture and workspace documents in this repository assume that file is
available locally before running the streaming replay pipeline.
